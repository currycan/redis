### 安装步骤
1. 运行：install.sh，界面会显示安装步骤

2. 安装：各节点依次选择1、2，安装ruby和redis

3. 配置和启动：此步骤需要在不同节点操作，配置完后启动服务

4. 创建master角色：任意节点运行，选择7

5. 创建slave角色：根据节点信息选择

6. 优化参数：各节点选择11

7. 其他说明：请根据实际情况选择安装脚本
- 本脚本可用于六节点集群和九节点集群，六节点集群11-13步骤不做
- 直接搭建九节点集群集群，节点关系可能会异常。建议先做完三主三从（六节点），插入一些数据，在加入剩下的三个节点


### 常见维护
```shell
# 查看集群状态：
redis-trib check <IP>:<PORT>
# 重新配置：
pkill -9 redis
make clean
rm -rf /data (根据需要)
```
