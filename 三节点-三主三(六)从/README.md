### 安装步骤
1. 运行：install.sh，界面会显示安装步骤

2. 安装：各节点依次选择1、2，安装ruby和redis

3. 配置和启动：此步骤需要在不同节点操作，配置完后启动服务（若集群起在同一节点，根据提示选择Y）
- 在node1上运行：依次选择3、6
- 在node2上运行：依次选择4、6
- 在node3上运行：依次选择5、6

4. 创建master角色：任意节点运行，选择7

5. 创建slave角色：根据节点信息选择，node1 选择8/node2 选择9/node3 选择10

6. 优化参数：各节点选择11

7. 其他说明：请根据实际情况选择安装脚本
- 三节点-三主三从, 直接按脚本install.sh提示运行即可
- 三节点-三主六从，先运行install.sh搭建三主三从脚本，然后用add_node脚本增加三个节点完成三主六从集群。


### 常见维护
```shell
# 查看集群状态：
redis-trib check <IP>:<PORT>
# 重新配置：   
pkill -9 redis
make clean
rm -rf /data (根据需要)
```
